# Borrador de solicitud a olzhass / OGAMES

Contacto público: `olzhass.carparking@gmail.com`

## English

**Subject:** Request for permission and technical documentation for an accessibility-focused CPM livery planning tool

Hello OGAMES / olzhass team,

I am developing a non-commercial experimental tool for the original Car Parking Multiplayer that helps players plan complex liveries using the game's normal visible editor. The tool is intended to generate a preview and a layer-by-layer recipe. It does not modify game files, memory, economy, servers, or anti-cheat systems.

Before continuing, I would like to ask whether you permit this type of manual-assistance/accessibility tool and whether you can provide any public or limited technical information about the livery editor, such as:

- decal/vinyl names or atlas intended for public use;
- layer limits for body and windows;
- X/Y/scale/rotation conventions;
- supported fonts and text symbols;
- grouping, mirror and copy behavior;
- a documented design import/export format, if one exists;
- permission conditions for using editor screenshots to build a visual calibration dataset.

We do not request source code or permission to modify the game. We will not extract or redistribute proprietary assets without explicit written authorization. We are willing to follow branding, attribution, rate, distribution and safety requirements you specify.

The goal is to make complex designs more accessible and reduce repetitive manual placement while keeping every action within the official editor.

Thank you for your time.

Sincerely,
[Name / project contact]
