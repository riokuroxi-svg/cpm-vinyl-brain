# Referencias externas estudiadas

La implementación CPM Vinyl Brain es propia. Se estudiaron los siguientes proyectos y técnicas:

- Optimize & Reduce — https://github.com/ajevnisek/optimize-and-reduce — MIT.
- Segmentation-Based Parametric Painting — https://github.com/manuelladron/semantic_based_painting — MIT.
- pycma — https://github.com/CMA-ES/pycma — BSD-3-Clause.
- Primitive — https://github.com/fogleman/primitive — licencia permisiva incluida en su repositorio.
- LIVE — https://github.com/Picsart-AI-Research/LIVE-Layerwise-Image-Vectorization.
- Layered Image Vectorization via Semantic Simplification — https://github.com/SZUVIZ/layered_vectorization.

No se redistribuye código de estos repositorios en la versión 0.5. Se adaptan conceptos de optimización, reducción, atención residual y progresión coarse-to-fine a máscaras fijas de CPM.
